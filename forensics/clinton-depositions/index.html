I. SIX QUANTITATIVE FRAMEWORKS

This analysis applies six quantitative frameworks consistently across both depositions:

Framework 1: Bayesian Inference (Individual Claims)

For each claim, we compute:

LR = P(Evidence | Deception) / P(Evidence | Truth)

Where LR > 1 means the evidence is more likely under deception than truth. The higher the LR, the stronger the evidence against the claim.

Posterior probability under each prior:

P(Truth | Evidence) = P(Truth) / [P(Truth) + P(Deception) x LR]

Framework 2: Joint Probability (Conjunction)

The defense requires ALL claims to be simultaneously true:

P(All True) = Product of P(Truth_i | Evidence_i)

Tested under three correlation models: independent, clustered, mega-clusters.

Framework 3: Fisher's Combined Test

chi-squared = -2 x Sum[ln(p_i)]

Where p_i = P(Truth_i | Evidence_i). Compared against chi-squared distribution with 2k degrees of freedom.

Framework 4: Binomial Distribution Test

Tests whether memory failures are randomly distributed or statistically targeted at topics with documentary contradictions.

Framework 5: Sensitivity Analysis

18 scenarios (3 priors x 3 correlation models x 2 deponents) testing whether conclusions survive extreme assumptions.

Framework 6: Behavioral Impossibility Test

Tests whether described behavior is consistent with the behavior of a genuinely innocent person who received the signals admitted under oath.

NEW Framework 7: Spousal Conjunction Test

Tests the probability that BOTH spouses are independently truthful when their testimonies contain internal contradictions with each other. This framework is unique to the Clinton Master analysis.
